💫 MATCH CALCULATOR (PYTHON TKINTER GUI)
=
🎯 PROJECT OVERVIEW
------------------------------
This is a fun and interactive mini project created using the Python **Tkinter** library.  
It calculates a "match percentage" between two names and displays a friendly message based on compatibility.

The project helps you explore the **Python GUI library (Tkinter)** while having fun creating a playful app.  
It’s simple, enjoyable, and great for beginners who want to learn about user interfaces and event-driven programming.

------------------------------
🎮 HOW IT WORKS
------------------------------
1️⃣ The user enters two names (e.g., your name and another person's name).

2️⃣ On clicking the "Calculate Match 🔮" button, the program combines both names.

3️⃣ Each character in the combined string is converted to its ASCII number.

4️⃣ The sum of these numbers is taken, and the remainder when divided by 101 gives a match score (0–100%).

5️⃣ A friendly or funny message is shown based on the result.
